2018年第20批领证通知（自1月18日开始领取）

近期，工业和信息化部批准同意了部分单位提出的电信网码号资源有关申请，现对2018年第20批《中华人民共和国电信网码号资源使用证书》颁发结果进行公示，并请相关单位尽快来人领取证书。现将有关事项通知如下： 

一、 领取要求： 

(一) 授权委托书样式.doc(34K)（样式附后）；  

(二) 领取人身份证复印件； 

(三) 领取人身份证原件； 

(四) 原《使用证书》（说明：证书换新，需将原证书交回）。 

l 特别提醒：一份证书对应一份授权委托书和领取人身份证复印件。

              授权委托书需盖申请单位公章。 

二、 领取时间：本批次领证时间自2019年1月18日起，每周一至周五，上午9:00-11:00，下午1:30-4:30。

三、 领取地址：北京市西城区月坛南街11号·中国信息通信研究院·信息通信业务行政许可综合受理厅。（月坛派出所对面）

四、 咨询电话：010-82140090。
